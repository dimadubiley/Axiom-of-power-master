# Аксиома Власти

> *Визуальная новелла об амбициях, власти и цене, которую за них платят.*

---

## О проекте

**Аксиома Власти** — текстовая визуальная новелла, разработанная на движке **Ren'Py**. Главная героиня — **Габриэла**, студентка из Гарварда, приехавшая по обмену в Оксфорд. С первых минут она оказывается в водовороте академических интриг, социальной иерархии и неожиданных отношений. Игрок сам решает, как она будет себя вести — и какую цену заплатит за власть.

Игра вдохновлена атмосферой DDLC и современных драматических VN.

---

## Персонажи

| Персонаж | Роль |
|----------|------|
| **Габриэла** | Главная героиня. Студентка Гарварда, умная и саркастичная. |
| **Мейсон** | Загадочный британец, один из центральных персонажей. |
| **Стейси** | Королева кампуса. Встречает Габриэлу с первых минут. |
| **Маргарет (Марго)** | Лучшая подруга Габриэлы, студентка Кембриджа. |
| **Оливия** | Одна из ключевых фигур оксфордского окружения. |
| **Боуи** | Ещё один обитатель этого мира со своей историей. |
| **Фрейя** | Дополнительный персонаж с собственной ролью в сюжете. |

## Геймплей

- **Выборы**, влияющие на развитие сюжета и отношения с персонажами
- **Система репутации и отношений** (переменные `stacey_rel`, `mason_rel`, `gabriela_reputation`)
- **Выбор комнаты** в начале игры, который меняет атмосферу прохождения
- **NVL-режим** для внутренних монологов и ключевых сцен
- **Галерея** (в разработке)
- **Changelog** с историей версий

## Запуск

1. Скачай и установи [Ren'Py SDK](https://www.renpy.org/latest.html)
2. Клонируй репозиторий:
   ```bash
   git clone https://github.com/dimadubiley/Axiom-of-power-master.git
   ```
3. Открой Ren'Py Launcher → **Browse Projects** → выбери папку проекта
4. Нажми **Launch Project**
